# gitbook-updated-cli

**gitbook-updated-cli** is a fork of deprecated [gitbook-cli](https://github.com/GitbookIO/gitbook-cli).

The purpose of the fork is to keep the dependencies up to date.

## How to install it?

```
$ npm install -g gitbook-updated-cli
```
